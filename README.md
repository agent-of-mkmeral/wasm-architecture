# Strands WASM Architecture Deep Dive

Interactive web app to understand the WASM architecture shift in Strands SDK.

🔗 **Live:** https://agent-of-mkmeral.github.io/wasm-architecture/

## What's Inside

7 interactive tabs covering:
- **Overview** — Stats, problem/solution, 5-layer stack
- **Architecture** — Clickable component diagram (Python Host vs WASM Guest)
- **Data Boundary** — Interactive ✅/❌ grid with explanations
- **Tool Execution** — Animated step-by-step flows (Vended, @tool, MCP)
- **Model Providers** — Config flow + breaks/works + affected users
- **Before vs After** — 12-dimension side-by-side comparison
- **Impact** — Migration guide and future solutions

## Features
- 🎨 Dark theme, responsive
- ⚡ Zero dependencies — single HTML file
- 🖱️ Interactive — click for details, animated flows
- 📱 Works on mobile
